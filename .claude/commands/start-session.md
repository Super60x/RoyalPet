# Start Sessie — RoyalPet.app

Je begint een nieuwe werksessie. Voer de volgende stappen uit:

## 1. Status ophalen
- Lees `CLAUDE.md` → sectie "Huidige Sessie" voor sessienummer en status
- Lees `checklist.md` → vind de eerste onafgevinkte items
- Scan `learnings/` → lees het meest recente bestand
- Run `git log --oneline -10` → bekijk laatste wijzigingen

## 2. Analyse
- Vat samen: wat is er vorige sessie gedaan?
- Wat zijn openstaande issues of blockers?
- Welke checklist-items zijn aan de beurt?

## 3. Plan voorstellen
- Stel een concreet plan voor deze sessie voor
- Geef prioriteit: wat levert het meeste op?
- Vraag bevestiging voordat je begint met bouwen

Begin NOOIT met coderen zonder eerst dit overzicht te geven en toestemming te vragen.
