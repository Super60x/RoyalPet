# Debug — RoyalPet.app

Er is een error. Volg dit stappenplan:

## 1. Error lezen
- Lees de VOLLEDIGE foutmelding (niet alleen de eerste regel)
- Identificeer: bestand, regelnummer, error type

## 2. Context verzamelen
- Lees het bestand waar de error optreedt
- Check `git diff` — welke recente wijzigingen kunnen dit veroorzaakt hebben?
- Check of het een bekende issue is in `learnings/`

## 3. Diagnose
- Leg uit WAT de error betekent in begrijpelijke taal
- Leg uit WAAROM het gebeurt
- Stel een fix voor

## 4. Fix & Test
- Pas de fix toe
- Test of de error weg is
- Test of er geen nieuwe errors zijn ontstaan

## 5. Documenteer
- Als het een niet-voor-de-hand-liggende fix was: sla op in `learnings/dag-XX.md`
- Categorie: bug fix + korte beschrijving

Leg bij elke stap uit wat je doet en waarom — dit is een leermoment.
