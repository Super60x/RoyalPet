# Learning Opslaan — RoyalPet.app

Sla een learning op voor toekomstige sessies.

## Stappen
1. Vraag wat er geleerd is (als niet meegegeven door de gebruiker)
2. Bepaal de huidige sessie uit `CLAUDE.md` → "Huidige Sessie"
3. Open of maak `learnings/dag-XX.md` aan (gebruik `learnings/_template.md` als basis)
4. Voeg de learning toe onder de juiste categorie:
   - **Bug fix** — wat ging er mis + hoe opgelost
   - **Architectuur** — waarom een bepaalde keuze gemaakt is
   - **Tool tip** — handige CLI commands, shortcuts, configuratie
   - **API gotcha** — onverwacht gedrag van Supabase/Stripe/Replicate/Resend
5. Bevestig aan de gebruiker dat het is opgeslagen
