# RoyalPet.app — Slash Commands

> Overzicht van alle beschikbare Claude Code slash commands voor dit project.

## Dagelijkse workflow

| Moment | Command | Wat het doet |
|---|---|---|
| Start sessie | `/start-session` | Status ophalen, analyseren, plan voorstellen |
| Tijdens sessie | `/debug` | Gestructureerd een error onderzoeken en fixen |
| Tijdens sessie | `/learn` | Een learning opslaan in learnings/dag-XX.md |
| Voor deploy | `/deploy-check` | Build, types, secrets en SEO controleren |
| Einde sessie | `/recap` | Samenvatten, checklist updaten, learnings opslaan |

## Typische sessie

```
1. /start-session          → Claude geeft overzicht + plan
2. Bevestig plan           → Begin met bouwen
3. /debug (indien nodig)   → Fix errors gestructureerd
4. /learn (indien nodig)   → Sla learnings op
5. /deploy-check           → Controleer voor deploy
6. /recap                  → Sluit sessie af
```
